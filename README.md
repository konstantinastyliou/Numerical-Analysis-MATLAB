# Numerical Analysis & Computational Algorithms in MATLAB

A comprehensive suite of numerical methods implemented in MATLAB, focusing on algorithmic efficiency, precision, and convergence analysis.

## Repository Structure

- **Approximations:** Taylor series expansion and error tracking for basic mathematical functions.
- **Root_Finding:** Non-linear equation solvers including Bisection, Fixed-Point Iteration, Newton-Raphson, and Secant methods.
- **Linear_Systems:** Linear system solvers featuring Gaussian Elimination with Partial Pivoting and the Jacobi Iterative Method.
- **Numerical_Integration:** Numerical integration techniques comparing Midpoint (Rectangle), Trapezoidal, and Simpson's rules with dynamic step-halving.

## How to Run
Every script is self-contained and can be executed directly within the MATLAB environment.
